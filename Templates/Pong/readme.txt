Controls
W -> Player1 UP
S -> Player1 DOWN
Arrow_Up -> Player2 UP
Arrow_Down -> Player2 DOWN
Escape -> Pause
Space -> Serve


Information
To start game use space;
The players have 3 lives (L);
The players have a board to bounce ball;
The players have a score;
The one who has more points wins;

It's using CharGraph++ for graphics;
Use line heigth of 0.8;
